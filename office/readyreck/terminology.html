Documentum Server

Definition: Documentum Server manages content in a repository. Documentum Server provides the essential capabilities for organizing, controlling, and accessing enterprise content stored within a repository. Documentum Content Server has been renamed as Documentum Server. Always use Documentum Server in product documentation. Do not use Documentum Content Server, Content Server, CS, and DS.

Correct Usage Incorrect Usage
Correct Documentum Server uses network locations to determine the content storage location from which a content file is provided to web client users. (Correct) Documentum Content Server uses network locations to determine the content storage location from which a content file is provided to web client users. Content Server uses network locations to determine the content storage location from which a content file is provided to web client users. CS uses network locations to determine the content storage location from which a content file is provided to web client users. (Incorrect)
